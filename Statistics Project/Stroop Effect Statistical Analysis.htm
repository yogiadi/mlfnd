Stroop Effect

Independent Variable:

Text name and colour are same(congruent) and different(incongruent).

Dependent Variable

Reaction time to read the name of colour in both cases: congruent and incongruent.

Null hypothesis:

The reaction time to read incongruent data is not greater than congruent data.

Alternative hypothesis:

The reaction time to read incongruent data is greater than congruent data.

Statistical Test:

A one tailed t-test with alpha level equal to 0.05 is performed because we just to need to prove if incongruent data takes more time to read than congruent data.

Assumptions:

1.    Provided are random samples from two independent variables.

2.    Populations are approximately normal.

3.    Sample data can estimate population variances.

4.    Population variances are roughly equal.

Descriptive Statistics:

Xcongruent = 14.0511

Xincongruent = 22.01592

Ncongruent = 24

Nincongruent = 24

 

SScongruent = 291.3877

SSincongruent = 529.2704

 

Pooled Variance , Sp2 = (SScongruent + SSincongruent )/(dfcongruent + dfincongruent )

 

where degree of freedom:

dfcongruent = Ncongruent -1=23

dfincongruent = Nincongruent -1=23

 

Hence , Sp2 = 17.84039

SEM = 1.219

tstatistics = ( Xincongruent - Xcongruent )/SEM = 6.532323

tcrtitical (at alpha level 0.05 for df=46)=1.684

P value is less than 0.0001.

Test Result:

Since p value is less than 0.0001 null hypothesis has been rejected.
